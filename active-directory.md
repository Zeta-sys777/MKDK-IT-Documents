# Active Directory

Этот раздел описывает домен Active Directory, пользователей, OU и группы.

Основные листы Excel:

- `it-templates_2026_04_06.xlsx` → `Пользователи`
- `it-templates_2026_04_06.xlsx` → `AD-группы`
- `it-templates_2026_04_06.xlsx` → `Сбросы паролей`

## Домен

Используется домен:

`MKDK`

## Базовая структура

В Excel указаны базовые OU:

- `Students`;
- `Teachers`;
- `Admins`.

Также указана группа:

- `IT-Department`.

## Пользователи

Информацию по пользователям пока оставляем внутри этого раздела.

Для создания пользователя нужны:

- ФИО;
- тип пользователя;
- группа;
- должность;
- логин;
- OU;
- группы AD;
- дата создания;
- кто создал;
- статус;
- комментарий.

Пароли в Excel и Markdown не хранятся.

## Группы AD

Группы AD используются для назначения прав.

В Excel нужно фиксировать:

- объект;
- тип;
- назначение;
- участников или область;
- права;
- статус;
- комментарий.

## Групповые политики

Групповые политики планируются к использованию.

После настройки нужно описать:

- название политики;
- к какой OU применяется;
- что именно настраивает;
- кто изменил;
- когда изменил;
- зачем была создана политика.

## Сброс паролей

Сброс пароля студенту выполняется только лично при предъявлении студенческого билета.

Пароль в журнале не хранится.

## Что нужно сделать

План работ:

1. Завершить структуру OU.
2. Заполнить группы AD.
3. Добавить пользователей студентов, преподавателей и администраторов.
4. Настроить групповые политики.
5. Описать правила логинов.
6. Описать правила отключения пользователей.
7. Проверить подключение учебных ПК к домену.
