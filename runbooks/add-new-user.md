# Инструкция: создание нового пользователя

## Где фиксировать

`it-templates_2026_04_06.xlsx` → `Пользователи`

## Кто выполняет

Создавать пользователей может только IT-департамент.

## Какие данные нужны

- ФИО.
- Тип пользователя.
- Группа.
- Должность.
- Логин.
- OU.
- Группы AD.
- Кто создал.
- Статус.

## Порядок действий

1. Получить основание для создания пользователя.
2. Проверить данные пользователя.
3. Создать учётную запись в Active Directory.
4. Назначить OU.
5. Добавить пользователя в нужные группы AD.
6. Заполнить лист `Пользователи`.
7. Добавить запись в журнал работ.

## Важно

Пароль не записывается в Excel и Markdown.
