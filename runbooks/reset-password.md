# Инструкция: сброс пароля

## Где фиксировать

`it-templates_2026_04_06.xlsx` → `Сбросы паролей`

## Кто может запросить сброс

Сброс пароля может запросить только сам пользователь лично.

## Проверка личности

Студент должен предъявить студенческий билет.

## Порядок действий

1. Принять личное обращение.
2. Проверить студенческий билет.
3. Найти пользователя в Active Directory.
4. Сгенерировать случайный временный пароль.
5. Выполнить сброс пароля.
6. Сообщить пароль пользователю лично.
7. Зафиксировать факт сброса в Excel.

## Важно

Пароль нельзя записывать в журнал.
