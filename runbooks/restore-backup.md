# Инструкция: восстановление из резервной копии

## Статус

Фактическая схема восстановления будет описана после настройки бэкапов.

## Общий порядок

1. Определить, что нужно восстановить.
2. Проверить доступные резервные копии.
3. Выбрать подходящую дату копии.
4. Согласовать восстановление.
5. Выполнить восстановление.
6. Проверить результат.
7. Зафиксировать работу в журнале.

## Важно

Нельзя восстанавливать данные поверх рабочих без понимания последствий.
