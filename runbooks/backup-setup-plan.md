# Инструкция: план настройки резервного копирования

## Цель

Настроить резервное копирование критичных объектов инфраструктуры.

## Что настроить

1. Бэкап виртуальной машины Windows Server.
2. Бэкап Active Directory.
3. Бэкап файлового сервера.
4. Бэкап сайта `mkdk-studsovet.ru`.
5. Бэкап конфигураций сетевого оборудования.

## Порядок работ

1. Выбрать хранилище.
3. Настроить копирование.
4. Проверить создание копий.
5. Выполнить тестовое восстановление.
6. Записать результат проверки в Excel.
7. Описать итоговую схему в `backup.md`.

## Важно

Бэкап без проверки восстановления не считается надёжным.
