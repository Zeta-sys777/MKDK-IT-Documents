# Карта репозитория

Этот файл показывает, где находится каждый раздел документации.

## Главные файлы

- `README.md` — главная страница и навигация.
- `overview.md` — общий обзор инфраструктуры.
- `inventory.md` — инвентаризация оборудования.
- `network.md` — сеть и IP-план.
- `servers.md` — серверы и виртуальные машины.
- `active-directory.md` — домен, пользователи, группы и OU.
- `backup.md` — план настройки резервного копирования.
- `security.md` — безопасность и правила доступа.
- `student-council-site.md` — сайт колледжа.
- `work-log.md` — журнал работ.

## Инструкции

- `runbooks/equipment-inventory.md` — проведение инвентаризации.
- `runbooks/join-pc-to-domain.md` — подключение ПК к домену.
- `runbooks/add-new-user.md` — создание пользователя.
- `runbooks/backup-setup-plan.md` — настройка бэкапов.
- `runbooks/clonezilla-deploy-images.md` — развёртывание образов через Clonezilla.
- `runbooks/restore-backup.md` — восстановление из резервной копии.
- `runbooks/work-log-rules.md` — правила ведения журнала работ.
