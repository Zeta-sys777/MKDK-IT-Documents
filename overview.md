Этот файл даёт краткое понимание текущего состояния IT-инфраструктуры колледжа.

## Основная информация

- Колледж: Московский колледж деловой карьеры.
- Адрес: Нижегородская 32СБ, Москва.
- Сайт: https://mkdk-studsovet.ru
- Домен Active Directory: `MKDK`.
- Серверная зона: 3 этаж.
- Wi-Fi: только для сотрудников.

## Что входит в инфраструктуру

В инфраструктуру входят:

- учебные компьютеры;
- компьютеры преподавателей;
- личные рабочие ПК участников IT-департамента;
- проектор;
- сервер Proxmox;
- виртуальная машина Windows Server;
- Active Directory;
- DNS;
- DHCP;
- будущий файловый сервер;
- сайт `mkdk-studsovet.ru`.
